# Greet Skill
The Greet skill is designed to provide a friendly and welcoming message to the user. It can be used in various contexts, such as when the user asks for a greeting, wants to be welcomed, or needs a motivational phrase.

## Usage
To use the Greet skill, simply invoke it when the user requests a greeting or a motivational message. The skill will respond with a warm and positive message to brighten the user's day.

## Example
User: "Hello, can you greet me?"
Agent: "Hello! It's great to see you. Remember, every day is a new opportunity to achieve your goals. Keep shining!"
