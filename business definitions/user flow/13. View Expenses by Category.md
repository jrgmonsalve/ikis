# UF-013: View Expenses by Category

## Related User Story

US-013: View Expenses by Category

## Goal

Allow a family member to understand how money is being spent by category.

## Main Flow

1. The family member opens the reports section.
2. The family member selects "Expenses by category".
3. The system shows the period filter.
4. The family member selects a period.
5. The period can be monthly, yearly, or custom date range.
6. The system loads expense transactions for the selected family and period.
7. The system groups expenses by category.
8. The system calculates the total spent per category.
9. The system shows the category name, total amount, color, and icon when available.
10. The system excludes income and transfers from the report.

## Alternative Flows

### Alternative Flow 1: No expenses for selected period

1. The family member selects a period without expenses.
2. The system shows an empty report state.
3. The system indicates that no expenses were found for the selected period.

### Alternative Flow 2: Invalid custom date range

1. The family member selects a custom date range.
2. The start date is after or equal to the end date.
3. The system does not load the report.
4. The system shows a validation message.

## Postconditions

* The family member can see expenses grouped by category.
* The report only includes expense transactions from the selected family and period.
