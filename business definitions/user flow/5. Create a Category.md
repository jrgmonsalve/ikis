# UF-005: Create a Category

## Related User Story

US-005: Create a Category

## Goal

Allow a family member to create a category for classifying transactions.

## Main Flow

1. The family member opens the categories section.
2. The system shows the list of existing categories.
3. The family member selects "Create category".
4. The system shows the category creation form.
5. The family member enters the category name.
6. The family member optionally selects a color.
7. The family member optionally selects an icon.
8. The family member submits the form.
9. The system validates the category information.
10. The system verifies that there is no active category with the same name in the selected family.
11. The system creates the category.
12. The system shows the new category in the categories list.
13. If an owner or admin edits a category, the system shows active subcategories for that category.
14. The owner or admin can create, rename, or deactivate subcategories inside that category.

## Alternative Flows

### Alternative Flow 1: Missing category name

1. The family member submits the form without a category name.
2. The system does not create the category.
3. The system shows a validation message.
4. The family member enters a valid category name.

### Alternative Flow 2: Duplicate category name

1. The family member enters a category name that already exists in the same family.
2. The system does not create the category.
3. The system shows a duplicate category message.
4. The family member changes the category name.

### Alternative Flow 3: Duplicate subcategory name

1. An owner or admin enters a subcategory name that already exists as active under the same category.
2. The system does not create or update the subcategory.
3. The system shows a duplicate subcategory message.
4. The owner or admin changes the subcategory name.

## Postconditions

* A new category exists inside the selected family.
* Active subcategories may exist under the category.
* The category can be used for income, expenses, reports, and budgets.
* The category may have a color and icon.
