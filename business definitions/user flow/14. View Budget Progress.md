# UF-014: View Budget Progress

## Related User Story

US-014: View Budget Progress

## Goal

Allow a family member to see whether the family is staying within planned spending limits.

## Main Flow

1. The family member opens the budgets section.
2. The system loads the budgets for the selected family.
3. The family member optionally filters budgets by period.
4. The system calculates the spent amount for each budget.
5. The system calculates the remaining amount for each budget.
6. The system calculates the percentage used.
7. The system shows each budget with its progress.
8. If a budget is exceeded, the system marks it as exceeded.

## Alternative Flows

### Alternative Flow 1: No budgets exist

1. The family member opens the budgets section.
2. The system detects that no budgets exist.
3. The system shows an empty state.
4. The system suggests creating the first budget.

### Alternative Flow 2: No expenses match a budget

1. The system calculates budget progress.
2. No expenses match the budget category and period.
3. The system shows spent amount as zero.
4. The system shows the full planned amount as available.

## Postconditions

* The family member can see budget usage.
* Each budget shows planned amount, spent amount, remaining amount, and percentage used.
* Exceeded budgets are clearly identified.
