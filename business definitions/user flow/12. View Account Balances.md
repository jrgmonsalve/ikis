# UF-012: View Account Balances

## Related User Story

US-012: View Account Balances

## Goal

Allow a family member to see the current balance of each family account and the total available money.

## Main Flow

1. The family member opens the accounts section or dashboard.
2. The system loads the accounts for the selected family.
3. The system calculates each account balance.
4. The system separates available money accounts from credit card accounts.
5. The system shows the current balance for each account.
6. The system calculates the total available balance.
7. The system shows credit card owed balances separately.

## Alternative Flows

### Alternative Flow 1: No accounts exist

1. The family member opens the accounts section.
2. The system detects that no accounts exist.
3. The system shows an empty state.
4. The system suggests creating the first account.

### Alternative Flow 2: Balance calculation fails

1. The system fails to calculate account balances.
2. The system shows an error message.
3. The user can retry later.

## Postconditions

* The family member can see current account balances.
* The total available balance excludes credit card owed balances.
