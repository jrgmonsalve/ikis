# US-009: Create a Budget

## Title

Create a budget

## User Story

As a family member, I want to create a budget for a specific period, so that the family can control spending limits according to monthly, yearly, or custom planning needs.

## Acceptance Criteria

* An authenticated family member can create a budget inside a family.
* The budget must have a name.
* The budget must be associated with a category.
* The budget must have a planned amount.
* The budget must use the active family period configured in Settings.
* The budget period type can be:

  * Monthly
  * Yearly
  * Custom date range
* If the active period type is monthly, the configured month and year are used.
* If the active period type is yearly, the configured year is used.
* If the active period type is custom date range, the configured start date and end date are used.
* The start date must be before the end date.
* The budget should calculate the spent amount using expenses from the selected category and selected period.
* The budget should show the remaining available amount.
* The same family should not have duplicate budgets for the same category and period.
* The budget should be used in reports and dashboard summaries.
* When a budget period has ended, an owner or admin can copy it to a new period.
* Copying preserves the budget name, category, planned amount, and currency.
* Copied budgets use the active family period configured in Settings.
* The copied budget starts with progress calculated only from expenses in its new period.
* Duplicate category-and-period validation also applies to copied budgets.

## Notes

Examples:

```
Budget Type: Monthly
Period: June 2026
Category: Food
Planned Amount: $1,000,000

Budget Type: Custom Date Range
Start Date: 2026-06-03
End Date: 2026-07-03
Category: Food
Planned Amount: $1,000,000

Budget Type: Yearly
Year: 2026
Category: Education
Planned Amount: $6,000,000
```
