# US-013: View Expenses by Category

## Title

View expenses by category

## User Story

As a family member, I want to view expenses grouped by category, so that I can understand where the family money is being spent.

## Acceptance Criteria

* An authenticated family member can view expenses grouped by category.
* The user can filter the report by period.
* The period can be monthly, yearly, or custom date range.
* Each category shows the total spent amount.
* Each category may show its assigned color and icon.
* The report should only include expense transactions.
* Income and transfers must not be included in this report.
* The report should be based on the selected family.

## Notes

Example:

```
Period: June 2026

Food: $650,000
Transport: $180,000
Utilities: $320,000
Health: $90,000
```
