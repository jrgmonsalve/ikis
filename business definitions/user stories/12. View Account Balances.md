# US-012: View Account Balances

## Title

View account balances

## User Story

As a family member, I want to view the current balance of each account, so that I can know how much money the family has available in real accounts.

## Acceptance Criteria

* An authenticated family member can view the list of family accounts.
* Each account shows its current balance.
* The current balance is calculated from the initial balance plus income, minus expenses, and adjusted by transfers.
* Credit card accounts should show the current owed amount or used balance.
* The user can see the total available balance across applicable accounts.
* The system should exclude credit card debt from available money calculations.
* The balances must reflect the latest registered transactions.

## Notes

Example:

```
Bancolombia Savings: $2,000,000
Nequi: $350,000
Cash: $120,000
Credit Card Visa: $800,000 owed

Total available: $2,470,000
```
