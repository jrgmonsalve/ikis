# US-011: Mark a Recurring Payment as Paid

## Title

Mark a recurring payment as paid

## User Story

As a family member, I want to mark a recurring payment as paid, so that the system registers the payment as an expense and updates the next due date.

## Acceptance Criteria

* An authenticated family member can mark an active recurring payment as paid.
* The user must confirm the payment amount.
* The user must select or confirm the account used to pay.
* The system creates an expense transaction using the recurring payment information.
* The selected account balance is reduced.
* The recurring payment status for the current period is marked as paid.
* The system calculates the next due date based on the recurring payment frequency.
* The created expense is included in reports and budget calculations.

## Notes

Example:

```
Recurring Payment: Internet
Expected Amount: $120,000
Account: Bancolombia Savings
Category: Utilities
Payment Date: 2026-06-15

Result:
Expense created
Account balance reduced
Next due date updated
```
