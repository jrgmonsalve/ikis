# US-001: Sign in with Google

## Title

Sign in with Google

## User Story

As a user, I want to sign in using my Google account, so that I can access the platform quickly without creating a separate username and password.

## Acceptance Criteria

* The user can access the app using a Google account.
* If the user signs in for the first time, the system creates a user profile.
* If the user already exists, the system loads the existing profile.
* The user must be authenticated before accessing family, accounts, transactions, budgets, or reports.
* The session should remain active unless the user signs out or the session expires.

## Notes

This story does not create a family yet. It only allows the user to enter the platform.
