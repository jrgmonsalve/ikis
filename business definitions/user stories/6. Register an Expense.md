# US-006: Register an Expense

## Title

Register an expense

## User Story

As a family member, I want to register an expense quickly, so that the account balance is updated and the family can track where money is being spent.

## Acceptance Criteria

* An authenticated family member can register an expense inside a family.
* The expense must have an amount.
* The expense must be associated with an account.
* The expense must be associated with a category.
* If the selected category has active subcategories, the expense must be associated with one active subcategory.
* If the selected category has no active subcategories, the expense is registered directly under the category.
* The expense may have a description.
* The expense may have a transaction date.
* If no transaction date is provided, the system uses the current date.
* An active manual expense can be edited by an active family member.
* Editing the expense keeps the same transaction and adjusts account balances as needed.
* An active expense can be cancelled by an active family member.
* Cancelling the expense keeps the transaction in history and rolls back the expense balance impact.
* When the expense is saved, the selected account balance is reduced.
* The expense is included in reports and budget calculations.

## Notes

Example:

```
Amount: $80,000
Account: Nequi
Category: Food
Subcategory: Groceries
Description: Groceries
Date: 2026-06-03
```
