# US-014: View Budget Progress

## Title

View budget progress

## User Story

As a family member, I want to view the progress of each budget, so that I can know whether the family is staying within the planned spending limits.

## Acceptance Criteria

* An authenticated family member can view active budgets inside a family.
* Each budget shows the planned amount.
* Each budget shows the spent amount.
* Each budget shows the remaining amount.
* Each budget shows the percentage used.
* The spent amount is calculated from expense transactions that match the budget category and period.
* Income and transfers must not affect budget usage.
* If the spent amount exceeds the planned amount, the budget should be marked as exceeded.
* The user can filter budgets by period.

## Notes

Example:

```
Budget: Food
Planned Amount: $1,000,000
Spent: $650,000
Remaining: $350,000
Used: 65%
```
