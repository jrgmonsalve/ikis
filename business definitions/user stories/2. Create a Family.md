# US-002: Create a Family

## Title

Create a family

## User Story

As a user, I want to create a family, so that I can manage personal or shared finances in a dedicated financial space.

## Acceptance Criteria

* The authenticated user can create a family.
* The user must provide a family name.
* The user who creates the family becomes the family owner.
* The family can be used by a single person or shared with other members later.
* After creating the family, the user can start creating accounts, categories, transactions, budgets, and recurring payments inside that family.

## Notes

This story only creates the family. It does not invite other members yet.
