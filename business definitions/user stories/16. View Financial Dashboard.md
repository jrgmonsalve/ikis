# US-016: View Financial Dashboard

## Title

View financial dashboard

## User Story

As a family member, I want to view a financial dashboard, so that I can quickly understand the family’s current financial status.

## Acceptance Criteria

* An authenticated family member can view the dashboard for the selected family.
* The dashboard shows total available balance.
* The dashboard shows total expenses for the selected period.
* The dashboard shows total income for the selected period.
* The dashboard shows budget progress summary.
* The dashboard shows upcoming recurring payments.
* The dashboard shows expenses by category.
* The user can change the dashboard period.
* The dashboard period can be monthly, yearly, or custom date range.
* The dashboard data must be based only on the selected family.

## Notes

Example:

```
Period: June 2026

Total Available Balance: $2,470,000
Income: $3,500,000
Expenses: $1,240,000
Budget Usage: 65%
Upcoming Payments: 3
Highest Expense Category: Food
```
