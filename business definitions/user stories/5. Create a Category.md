# US-005: Create a Category

## Title

Create a category

## User Story

As a family member, I want to create expense and income categories, so that transactions can be classified and later analyzed in reports and budgets.

## Acceptance Criteria

* An authenticated family member can create a category inside a family.
* The category must have a name.
* The category may have a color.
* The category may have an icon.
* The category must belong to one family.
* Owner and admin users can manage active subcategories inside an existing category.
* A subcategory must have a name and belongs to one parent category.
* A category cannot have two active subcategories with the same name.
* The category can be used when registering income or expenses.
* The same family should not have duplicate category names.
* Categories can be used later for reports and budgets.

## Notes

Examples:

```
Food
Food > Groceries
Food > Fast food
Transport
Transport > Public transport
Transport > Taxi
Utilities
Salary
Education
Health
```
