# US-004: Create an Account

## Title

Create an account

## User Story

As a family member, I want to create a financial account, so that I can track where the family money exists or is owed.

## Acceptance Criteria

* An authenticated family member can create an account inside a family.
* The account must have a name.
* The account must have an account type.
* The account may have an initial balance.
* The account belongs to one family.
* The account can be used later when registering income, expenses, or transfers.

## Account Types

* Savings
* Cash
* Digital Wallet
* Credit Card

## Notes

This story only creates the account. It does not register transactions yet.
