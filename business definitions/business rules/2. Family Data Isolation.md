# BR-002: Family Data Isolation

## Title

Family data isolation

## Business Rule

A user can only access financial data that belongs to a family where the user is an active member.

## Details

* A user cannot access a family unless they are a member of that family.
* A user cannot access accounts from another family.
* A user cannot access categories from another family.
* A user cannot access transactions from another family.
* A user cannot access budgets from another family.
* A user cannot access recurring payments from another family.
* Reports and dashboards must only use data from the selected family.

## Reason

Each family represents an isolated financial workspace.
