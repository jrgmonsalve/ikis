# BR-001: Authenticated Access Required

## Title

Authenticated access required

## Business Rule

A user must be authenticated before accessing any family financial information.

## Details

* Unauthenticated users cannot access families.
* Unauthenticated users cannot access accounts.
* Unauthenticated users cannot access categories.
* Unauthenticated users cannot access transactions.
* Unauthenticated users cannot access budgets.
* Unauthenticated users cannot access recurring payments.
* Unauthenticated users cannot access reports or dashboards.
* all except ... pending to define, in this moment all

## Reason

Financial information is private and must only be visible to authenticated users.
